# Wi-Fi QR Code Generator React-Native App
Using "WIFI:T:WPA;S:${wifiName};P:${wifiPassword};H:false;;" code and autoConnect Wi-Fi.

# Support
Network connection with QR Code is exclusive to iPhones only.
